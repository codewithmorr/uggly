This is a flask based e-commerce platform where product browsing and adding products to cart is possible 

  FEATURES

1. product browsing
2. cart holding feature
3. calculations of totals
4. size and quantity selection per product

   TECH STACK

 1. backend   - flask(python)
 2. frontend  - HTML/CSS(bootstrap)
 3. Database  - flask(python)
  
   SETUP INSTRUCTIONS
STEP 1 - CLONE REPOSITORY

  git clone https://github.com/codewithmorr/uggly.git
cd uggly

STEP 2 - CREATING VIRTUAL ENVIRONMENT 

python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows


STEP 3 - INSTALL DEPENDANCIES


  pip install -r requirements.txt

  
STEP 4 -RUN THE APPLICATION


  flask run
 or 
 python.app.py
